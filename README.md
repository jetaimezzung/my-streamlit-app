# 🐄 COW (Context Over Words)

## 📌 프로젝트 소개

COW는 사용자가 **지금 곧 말해야 하는 상황**을 자유롭게 입력하면,  
그 맥락에 맞는 **최소한의 생존 발화(Survival Line)**를 즉시 제공하는  
실전 중심 언어 훈련 Streamlit 앱입니다.

기존 스피킹 앱처럼 선택지를 고르거나 긴 설명을 학습하지 않고,  
실제 대면 상황에서 바로 쓸 수 있는 문장 하나에 집중합니다.
---
## ⭐ 핵심 기능 (MVP)

1. **자유 맥락 입력**  
   - 사용자가 선택 없이 자연어로 상황을 직접 입력

2. **AI 핵심 맥락 요약**  
   - 입력된 상황을 한 줄로 정리하여 현재 맥락을 명확히 인식

3. **Survival Line 즉시 제시**  
   - 설명 없이 바로 사용할 수 있는 핵심 문장 1개 제공

4. **말해보기 + 최소 피드백**  
   - 전달 여부, 길이 적절성 등 즉각적인 체크 중심 피드백

5. **누적 연습 기록 자동 저장**  
   - 별도 리포트 없이 백그라운드에서 사용 패턴 누적
---

## 🧭 사용 방법

1. 앱 실행
2. “지금 곧 말해야 하는 상황”을 한 줄로 입력
3. 상황 요약 확인 후 바로 진행
4. 제시된 문장을 말해보고 피드백 확인
